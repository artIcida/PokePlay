# PokéPlay
Este proxecto é unha plataforma para ver os capítulos da serie orixinal de Pokémon dobrados ao castelán de España.
Nós non almacenamos a media, tan só facilitamos de forma ordenada e accesíbel un contido que xa está na web.
Podes acceder en liña ao **PokéPlay** [premendo aquí](https://articida.github.io/PokePlay/)
